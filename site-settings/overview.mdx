### Name
The name of your site that will be used to help idenify your site in the Siimple dashboard, browser tabs and for SEO tags.

<Note>
  You can define a custom SEO title in the <a href="/site-settings/seo-settings">SEO settings page</a>.
</Note>

### Description

The description of your site that will be used for SEO tags.

<Note>
  You can define a custom SEO description in the <a href="/site-settings/seo-settings">SEO settings page</a>.
</Note>

### Website Email

The email address that will be used for contact forms that are added to this site and other email related features.

### Website Domain

The domain of your site where it will be publicly accessible on the web once published. For more information on how to set up your domain:

### Update Automatically Your Site?

If you enable this option, your site will be automatically when new versions of your chosen theme are available. This is a great way to keep your site up-to-date with the latest features and bug fixes.
